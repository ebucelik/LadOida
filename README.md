# LadOida
Is an iOS app which helps looking for charging stations for electro vehicles in Austria.
The app shows the closest charging stations nearby the given address or location.

Technologies that were used:
- SwiftUI
- Swift
- MapKit
- CoreLocation
- TCA
- Lottie
- SwiftMessages
- REST API
